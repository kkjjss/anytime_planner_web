# 애니타임 플래너
⏰반복되는 일정 관리에 최적화된 플래너 웹서비스⏰

## 기능
특정 하루, 기간뿐만 아니라 다양한 타입의 반복되는 일정을 손쉽게 등록할 수 있다.

### 단순 지정
- 하루 혹은 기간

### 반복
1. ~마다 반복: 매일, 매주, 매달, 매년, N일(ex. 10일에 한번) 등
2. ~동안 반복: 주, 달, 년, 기간
